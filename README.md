## Student Performance Analysis and Prediction: A Machine Learning Project

This project analyzes how various factors affect student performance in exams, measured by test scores. Factors such as Gender, Ethnicity, Parental Level of Education, Lunch Type, and Test Preparation Course are considered. The goal is to provide insights and predict student performance based on these variables.

--- 
## Dataset Description

- **Source**: [Kaggle - Students Performance in Exams](https://www.kaggle.com/datasets/spscientist/students-performance-in-exams?datasetId=74977)
- **Size**: 8 columns and 1000 rows
- **Columns**:
  - `gender`: Gender of the student
  - `race/ethnicity`: Student's ethnic group
  - `parental level of education`: Highest educational qualification of the parents
  - `lunch`: Type of lunch the student receives (e.g., standard or free/reduced)
  - `test preparation course`: Whether the student completed a test preparation course
  - `math score`: Score in Mathematics
  - `reading score`: Score in Reading
  - `writing score`: Score in Writing
